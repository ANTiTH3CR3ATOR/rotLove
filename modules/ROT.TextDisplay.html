TextDisplay:clear ( [c=' ' [, x=1 [, y=1 [, w [, h [, fg [, bg]]]]]]])
Clear the screen. By default wipes the screen to the default background color. You can provide a character, x-position, y-position, width, height, fore-color and back-color and write the same character to a portion of the screen

Parameters:

  • c string A character to write to the screen - may fail for strings with a length > 1
  • x int The x-position from which to begin the wipe
  • y int The y-position from which to begin the wipe
  • w int The number of chars to wipe in the x direction
  • h int Then number of chars to wipe in the y direction
  • fg table The color used to write the provided character
  • bg table the color used to fill in the background of the cleared space
